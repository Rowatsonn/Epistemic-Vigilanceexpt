Instructions

In this study, you will be shown a series of pictures like the one below. Your task is to guess the number of BLUE circles minus the number of YELLOW circles. You will enter your answer using a slider. If you thought there were 2 more BLUE circles then YELLOW circles, you would answer 2. Whereas, if you thought there were 2 fewer BLUE circles than YELLOW circles, you would answer -2.

You will only be given 2 seconds to view each picture, so make sure you are ready before beginning each question.

For example, here there are 8 BLUE circles and 6 YELLOW circles, so the correct answer is 2. You do not need to be precise to get the question correct. Your answer will be counted as correct if it is within +1 or -1 of the correct answer. Here, answering 1, 2 or 3 would be marked as a correct answer.

You will complete the study in a pair with another participant. You will either be player A or player B.

On each question, player A answers first, then player B answers. Player A’s task will be easier than player B’s task, but the correct answer will be the same for both players. This is done by increasing the number of circles on screen at a time for player B.

On the next page, you will see examples of Player A questions and a Player B questions.
